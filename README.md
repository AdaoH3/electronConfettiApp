 # Electron JS file code for Timer Interface 
 # Works with https://github.com/AdaoH3/timerControl
1. Takes User Input for defined time in 00:00:00 (Hours, minutes, seconds)
2. Sends API POST Requests through http://localhost:5000
   function startClicks(clickTimeStart): Format seconds: clickTimeStart ->  Server.py
   
3. Starts a visual timer on the screen in format 00:00:00
